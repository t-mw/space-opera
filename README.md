## Build

1. Activate Emscripten
2. Build [`libraylib.bc`](https://github.com/raysan5/raylib/wiki/Working-for-Web-(HTML5)) and extract to `./lib`
3. `make assets -B && make watch`
4. Open http://localhost:8000
